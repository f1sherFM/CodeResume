Vowel Count {{ kyu }}

Напишите функцию, которая возвращает количество гласных в заданной строке.

  • Гласными считаются буквы a, e, i, o, u
  • Функция должна быть регистронезависимой
  • Должна обрабатывать пустую строку
  • Должна игнорировать другие символы
View on Codewars

My Solution:

def vowel_count(s):
    """Return the number of vowels in a string."""
    vowels = 'aeiou'
    return sum(1 for char in s.lower() if char in vowels)

Test Examples:

# Basic case
vowel_count('abracadabra')  # Expected: 5

# No vowels
vowel_count('my pyx')       # Expected: 0

# Mixed case
vowel_count('AEIOU')        # Expected: 5

# Empty string
vowel_count('')             # Expected: 0

# With numbers/symbols
vowel_count('123!@#')       # Expected: 0
