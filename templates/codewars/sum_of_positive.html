Sum of Positive {{ kyu }}

Напишите функцию для суммирования положительных чисел в массиве. Условия:

  • Возвращает сумму всех положительных чисел
  • Игнорирует отрицательные числа
  • Возвращает 0 если массив пуст
  • Возвращает 0 если нет положительных чисел
  • Работает с целыми и дробными числами
View on Codewars

My Solution:

def positive_sum(arr):
    """Returns sum of all positive numbers in array."""
    return sum(x for x in arr if x > 0)

Test Examples:

# Basic case
positive_sum([1, 2, 3, 4, 5])      # Expected: 15

# With negatives
positive_sum([-1, 2, -3, 4, -5])   # Expected: 6

# Empty array
positive_sum([])                    # Expected: 0

# All negatives
positive_sum([-1, -2, -3])          # Expected: 0

# Mixed numbers
positive_sum([-1.5, 2.3, -3, 4.7])  # Expected: 7.0

# Single element
positive_sum([5])                   # Expected: 5
