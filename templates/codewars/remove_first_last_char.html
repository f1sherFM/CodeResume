Remove First and Last Character {{ kyu }}

Напишите функцию, которая удаляет первый и последний символ строки.

  • Работает со строками длиной ≥ 2
  • Не изменяет пустые строки
  • Возвращает пустую строку для ввода из 1-2 символов
View on Codewars

My Solution:

def remove_char(s):
    """Remove first and last character from string."""
    return s[1:-1]

Test Examples:

# Basic case
remove_char('eloquent')  # Expected: 'loquen'

# Short strings
remove_char('country')   # Expected: 'ountr'
remove_char('ok')        # Expected: ''
remove_char('a')         # Expected: ''

# Edge cases
remove_char('')          # Expected: ''
