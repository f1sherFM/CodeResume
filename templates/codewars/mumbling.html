Mumbling {{ kyu }}

Преобразуйте строку по шаблону, где каждая буква повторяется столько раз, каков ее индекс:

  • Первая буква в верхнем регистре
  • Последующие буквы в нижнем регистре
  • Буквы разделены дефисами
View on Codewars

My Solution:

def accum(s):
    """Convert string to mumbling pattern."""
    return '-'.join(c.upper() + c.lower() * i for i, c in enumerate(s))

Test Examples:

# Basic case
accum("abcd")    # Expected: "A-Bb-Ccc-Dddd"

# Mixed case input
accum("RqaEzty") # Expected: "R-Qq-Aaa-Eeee-Zzzzz-Tttttt-Yyyyyyy"

# Single character
accum("Z")       # Expected: "Z"

# With numbers/symbols
accum("cwAt")    # Expected: "C-Ww-Aaa-Tttt"
