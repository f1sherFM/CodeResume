Multiples of 3 or 5 {{ kyu }}

Найдите сумму всех чисел, кратных 3 или 5, ниже переданного числа.

  • Если число отрицательное, возвращает 0
  • Числа кратные и 3, и 5 учитываются один раз
  • Использует строгое неравенство (меньше переданного числа)
View on Codewars

My Solution:

def solution(number):
    """Return sum of all multiples of 3 or 5 below number."""
    return sum(x for x in range(number) if x % 3 == 0 or x % 5 == 0)

Test Examples:

# Basic case
solution(10)  # Expected: 23 (3+5+6+9)

# With zero
solution(0)   # Expected: 0

# Negative number
solution(-5)  # Expected: 0

# Large number
solution(20)  # Expected: 78 (3+5+6+9+10+12+15+18)
