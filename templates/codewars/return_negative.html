Return Negative {{ kyu }}

Напишите функцию, которая возвращает отрицательное число для положительного ввода, иначе возвращает само число.

  • Если число положительное - вернуть отрицательное
  • Если число отрицательное - вернуть без изменений
  • Если 0 - вернуть 0
View on Codewars

My Solution:

def make_negative(number):
    """Return negative of positive number, else return number."""
    return -number if number > 0 else number

Test Examples:

# Positive number
make_negative(1)     # Expected: -1

# Negative number
make_negative(-5)    # Expected: -5

# Zero
make_negative(0)     # Expected: 0

# Floating point
make_negative(0.12)  # Expected: -0.12

# Already negative
make_negative(-0.5)  # Expected: -0.5
