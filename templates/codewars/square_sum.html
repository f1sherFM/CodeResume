Square(n) Sum {{ kyu }}

Возведите в квадрат каждый элемент массива и суммируйте результаты:

  • Работает с целыми и дробными числами
  • Для пустого массива возвращает 0
  • Игнорирует нечисловые элементы
View on Codewars

My Solution:

def square_sum(numbers):
    """Return sum of squared numbers."""
    return sum(x ** 2 for x in numbers)

Test Examples:

# Basic case
square_sum([1, 2])               # Expected: 5
# With zeros
square_sum([0, 3, 4, 5])         # Expected: 50
# Empty array
square_sum([])                    # Expected: 0
# With negatives
square_sum([-1, -2])              # Expected: 5
