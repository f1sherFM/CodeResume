Grasshopper - Summation {{ kyu }}

Найдите сумму всех чисел от 1 до заданного числа n:

  • Число всегда положительное
  • Если n = 1, возвращает 1
  • Должен работать с большими числами
View on Codewars

My Solution:

def summation(num):
    """Return the sum of all numbers from 1 to num."""
    return sum(range(1, num + 1))

Test Examples:

# Small number
summation(1)    # Expected: 1

# Medium number
summation(8)    # Expected: 36 (1+2+3+4+5+6+7+8)

# Larger number
summation(100)  # Expected: 5050

# Edge case
summation(0)    # Expected: 0
