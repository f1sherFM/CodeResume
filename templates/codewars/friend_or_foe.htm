Friend or Foe? {{ kyu }}

Отфильтруйте массив строк, оставив только строки длиной 4 символа:

  • Возвращает новый массив
  • Не изменяет исходный массив
  • Работает с любыми строками
View on Codewars

My Solution:

def friend(friends):
    """Filter list to keep only names with 4 letters."""
    return [name for name in friends if len(name) == 4]

Test Examples:

# Basic case
friend(["Ryan", "Kieran", "Mark"])  # Expected: ["Ryan", "Mark"]
# No matches
friend(["Jimm", "Cari", "aret"])    # Expected: []
# With numbers
friend(["1234", "123", "12345"])    # Expected: ["1234"]
