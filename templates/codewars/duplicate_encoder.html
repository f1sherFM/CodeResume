Duplicate Encoder {{ kyu }}

Преобразуйте строку, заменяя символы:

  • '(' если символ встречается один раз
  • ')' если символ повторяется
  • Регистр не имеет значения
View on Codewars

My Solution:

def duplicate_encode(word):
    """Convert string to duplicate encoded format."""
    word = word.lower()
    return ''.join('(' if word.count(c) == 1 else ')' for c in word)

Test Examples:

# No duplicates
duplicate_encode("din")      # Expected: "((("
# With duplicates
duplicate_encode("recede")   # Expected: "()()()"
# Mixed case
duplicate_encode("Success")  # Expected: ")())())"
# Special chars
duplicate_encode("(( @")     # Expected: "))(("
