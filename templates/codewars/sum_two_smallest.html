Sum of two lowest positive integers {{ kyu }}

Найдите сумму двух наименьших положительных чисел в массиве:

  • Массив содержит минимум 4 положительных числа
  • Игнорирует отрицательные числа
  • Работает с большими числами
View on Codewars

My Solution:

def sum_two_smallest_numbers(numbers):
    """Return sum of two smallest numbers."""
    sorted_numbers = sorted([x for x in numbers if x > 0])
    return sorted_numbers[0] + sorted_numbers[1]

Test Examples:

# Basic case
sum_two_smallest_numbers([5, 8, 12, 18, 22])  # Expected: 13
# With large numbers
sum_two_smallest_numbers([10, 343445353, 3453445, 3453545353453])  # Expected: 3453455
# With negatives
sum_two_smallest_numbers([-1, 2, 3, 4])       # Expected: 5
