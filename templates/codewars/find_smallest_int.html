Find the smallest integer in the array {{ kyu }}

Найдите наименьшее целое число в заданном массиве:

  • Массив гарантированно не пустой
  • Может содержать отрицательные числа
  • Должен возвращать наименьшее число
View on Codewars

My Solution:

def find_smallest_int(arr):
    """Return the smallest integer in the array."""
    return min(arr)

Test Examples:

# With negative numbers
find_smallest_int([34, -345, -1, 100])  # Expected: -345

# All positive numbers
find_smallest_int([8, 10, 15, 20])      # Expected: 8

# Single element array
find_smallest_int([5])                   # Expected: 5

# Mixed numbers
find_smallest_int([0, 100, -5, 10])     # Expected: -5
