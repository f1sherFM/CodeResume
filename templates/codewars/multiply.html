Multiply {{ kyu }}

Напишите функцию, которая принимает два числа и возвращает их произведение. Функция должна:

  • Корректно работать с целыми числами и числами с плавающей точкой
  • Обрабатывать умножение на ноль
  • Учитывать знаки чисел (положительные/отрицательные)
  • Возвращать 0 при умножении любого числа на ноль
View on Codewars

My Solution:

def multiply(a, b):
    """Returns the product of two numbers."""
    return a * b

Test Examples:

# Basic multiplication
multiply(2, 3)    # Expected: 6

# With zero
multiply(5, 0)    # Expected: 0

# Negative numbers
multiply(-3, 4)   # Expected: -12

# Floating point
multiply(2.5, 4)  # Expected: 10.0

# Commutative property
multiply(4, 2.5)  # Expected: 10.0
