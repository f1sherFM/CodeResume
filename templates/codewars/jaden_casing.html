Jaden Casing Strings {{ kyu }}

Преобразуйте строку так, чтобы каждое слово начиналось с заглавной буквы:

  • Должен работать с любыми словами
  • Сохраняет пробелы между словами
  • Обрабатывает специальные символы
View on Codewars

My Solution:

def to_jaden_case(string):
    """Convert string to Jaden Case (each word capitalized)."""
    return ' '.join(word.capitalize() for word in string.split())

Test Examples:

# Basic case
to_jaden_case("how can mirrors be real")  # Expected: "How Can Mirrors Be Real"

# With apostrophes
to_jaden_case("if our eyes aren't real")  # Expected: "If Our Eyes Aren't Real"

# Single word
to_jaden_case("hello")                   # Expected: "Hello"

# Mixed case input
to_jaden_case("tHis iS a tEst")         # Expected: "This Is A Test"
