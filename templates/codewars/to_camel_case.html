Convert string to camel case {{ kyu }}

Преобразуйте строки с разделителями '-' или '_' в camelCase:

  • Первое слово сохраняет исходный регистр
  • Последующие слова начинаются с заглавной буквы
  • Удаляет все разделители
View on Codewars

My Solution:

def to_camel_case(text):
    """Convert dash/underscore delimited words to camelCase."""
    words = text.replace('-', ' ').replace('_', ' ').split()
    return words[0] + ''.join(word.capitalize() for word in words[1:])

Test Examples:

# Dash separated
to_camel_case("the-stealth-warrior")  # Expected: "theStealthWarrior"
# Underscore separated
to_camel_case("The_Stealth_Warrior")  # Expected: "TheStealthWarrior"
# Mixed delimiters
to_camel_case("A-B_C-D")              # Expected: "ABCD"
