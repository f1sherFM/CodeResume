Tribonacci Sequence {{ kyu }}

Реализуйте последовательность Трибоначчи (каждый элемент - сумма трех предыдущих):

  • Начальные 3 элемента задаются
  • Для n=0 вернуть пустой массив
  • Работает с целыми и дробными числами
View on Codewars

My Solution:

def tribonacci(signature, n):
    """Generate Tribonacci sequence."""
    sequence = signature[:n]
    for i in range(3, n):
        next_val = sum(sequence[i-3:i])
        sequence.append(next_val)
    return sequence

Test Examples:

# Basic case
tribonacci([1, 1, 1], 10)  # Expected: [1, 1, 1, 3, 5, 9, 17, 31, 57, 105]
# With zeros
tribonacci([0, 0, 1], 10)  # Expected: [0, 0, 1, 1, 2, 4, 7, 13, 24, 44]
# Empty result
tribonacci([1, 2, 3], 0)   # Expected: []
