String Repeat {{ kyu }}

Напишите функцию, которая повторяет строку указанное количество раз.

  • Принимает целое число ≥ 0
  • Возвращает пустую строку при repeat = 0
  • Сохраняет исходный регистр символов
View on Codewars

My Solution:

def repeat_str(repeat, string):
    """Repeat string specified number of times."""
    return string * repeat

Test Examples:

# Basic case
repeat_str(3, 'hi')      # Expected: 'hihihi'

# With zero
repeat_str(0, 'Hello')   # Expected: ''

# Single repeat
repeat_str(1, 'test')    # Expected: 'test'

# Long string
repeat_str(2, 'abc123')  # Expected: 'abc123abc123'
