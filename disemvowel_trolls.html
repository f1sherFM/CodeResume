Disemvowel Trolls {{ kyu }}

Напишите функцию, которая удаляет все гласные из строки.

  • Удаляет как строчные, так и заглавные гласные
  • Должна сохранять все остальные символы
  • Должна обрабатывать пустую строку
  • Должна сохранять пробелы и знаки препинания
View on Codewars

My Solution:

def disemvowel(s):
    """Remove all vowels from a string."""
    vowels = 'aeiouAEIOU'
    return ''.join(char for char in s if char not in vowels)

Test Examples:

# Basic case
disemvowel('This website is for losers LOL!')  # Expected: 'Ths wbst s fr lsrs LL!'

# Empty string
disemvowel('')                                 # Expected: ''

# All vowels
disemvowel('AEIOUaeiou')                       # Expected: ''

# Mixed case with newlines
disemvowel('No offense but,\\nYour writing is terrible.')  # Expected: 'N ffns bt,\\nYr wrtng s trrbl.'
