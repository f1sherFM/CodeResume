Reversed Strings {{ kyu }}

Напишите функцию, которая переворачивает переданную строку.

  • Должна обрабатывать пустую строку
  • Должна сохранять регистр символов
  • Должна работать с однобайтовыми символами
View on Codewars

My Solution:

def solution(string):
    """Reverse the given string."""
    return string[::-1]

Test Examples:

# Basic case
solution('world')    # Expected: 'dlrow'

# Empty string
solution('')         # Expected: ''

# Single character
solution('a')        # Expected: 'a'

# Palindrome
solution('racecar')  # Expected: 'racecar'

# With spaces
solution('hello world')  # Expected: 'dlrow olleh'
