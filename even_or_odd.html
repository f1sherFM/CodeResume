Even or Odd {{ kyu }}

Создайте функцию, которая определяет чётность целого числа. Требования:

  • Возвращает строку "Even" для чётных чисел
  • Возвращает строку "Odd" для нечётных чисел
  • Ноль должен считаться чётным числом
  • Должна работать с отрицательными числами
  • Принимает только целые числа
View on Codewars

My Solution:

def even_or_odd(number):
    """Determines if a number is even or odd."""
    return 'Even' if number % 2 == 0 else 'Odd'

Test Examples:

# Even numbers
even_or_odd(4)     # Expected: "Even"
even_or_odd(0)     # Expected: "Even"
even_or_odd(-2)    # Expected: "Even"

# Odd numbers
even_or_odd(7)     # Expected: "Odd"
even_or_odd(-3)    # Expected: "Odd"

# Edge cases
even_or_odd(1)     # Expected: "Odd"
even_or_odd(100)   # Expected: "Even"
